== Description ==

Buddypress show logged in user images


== Installation ==

1. Upload `vij-logged-in-buddypress.php` to the `/wp-content/plugins/` directory
2.  Single-site BuddyPress go to Plugins menu and activate there.
3.  For Multisite visit Network Admin -> Plugins and Network Activate it there.
4. Place `<?php do_action('plugin_name_hook'); ?>` in your templates
